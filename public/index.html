File Metadata Microservice

User stories:

  • When I submit a file, I will receive the file name, size in bytes, encoding, and mimetype within the JSON response.

Example usage :

Just choose a file and submit.If it's a large file, give it some time to upload.

Example output:

{"name":"attachment","size":100604,"encoding":"7bit","mimetype":"application/octet-stream"}
